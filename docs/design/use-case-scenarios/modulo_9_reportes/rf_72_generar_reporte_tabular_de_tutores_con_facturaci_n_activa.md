# RF-72: Generar reporte tabular de tutores con facturación activa

| Campo | Descripción |
| :---: | :---- |
| **Autor** |   |
| **Nombre** | RF-72: Generar reporte tabular de tutores con facturación activa |
| **Actor** | Administrador y Gestor Administrativo |
| **Objetivo** | Permitir que el actor genere y consulte un reporte tabular con todos los tutores que tienen activa la bandera de requerimiento de facturación externa y cuentan con datos fiscales registrados. |
| **Flujo Principal** | 1. El actor ingresa al módulo Directorio de datos fiscales.<br>2. El sistema muestra las opciones disponibles del directorio fiscal.<br>3. El actor selecciona la opción "Generar reporte de tutores con facturación activa".<br>4. El sistema valida los permisos del actor para consultar información fiscal.<br>5. El sistema consulta los tutores que tienen activa la bandera "Requiere facturación externa".<br>6. El sistema recupera los datos fiscales asociados a cada tutor encontrado.<br>7. El sistema consolida la información en una estructura tabular.<br>8. El sistema genera el reporte mostrando nombre del tutor, RFC, régimen fiscal, código postal y CFDI de uso.<br>9. El sistema muestra el reporte en pantalla.<br>10. El actor selecciona la opción Exportar.<br>11. El actor elige el formato de exportación (PDF o Excel).<br>12. El sistema genera el archivo correspondiente.<br>13. El sistema descarga o pone a disposición el archivo generado. |
| **Flujo Alterno** | **A. Usuario sin permisos suficientes**<br>&nbsp;&nbsp;&nbsp;&nbsp;1. En el paso 4, el sistema detecta que el actor no cuenta con permisos para consultar información fiscal.<br>&nbsp;&nbsp;&nbsp;&nbsp;2. El sistema muestra el mensaje: **"No cuenta con permisos para realizar esta operación"**.<br>&nbsp;&nbsp;&nbsp;&nbsp;3. El caso de uso termina. **B. No existen tutores con facturación activa**<br>&nbsp;&nbsp;&nbsp;&nbsp;1. En el paso 5, el sistema no encuentra tutores con la bandera de facturación externa activa.<br>&nbsp;&nbsp;&nbsp;&nbsp;2. El sistema muestra el mensaje: **"No existen tutores con requerimiento de facturación activo"**.<br>&nbsp;&nbsp;&nbsp;&nbsp;3. El caso de uso termina. |
| **Precondiciones** | El actor debe tener sesión activa. Deben existir tutores registrados en el sistema |
| **Postcondiciones** | El sistema genera y muestra el reporte de tutores con facturación activa. Opcionalmente, el reporte puede exportarse a un formato externo. |
| **Reglas de negocio involucradas** | ·         Solo deben incluirse en el reporte los tutores que tengan activa la bandera **"Requiere facturación externa"**. ·         El reporte debe mostrar únicamente la información fiscal vigente registrada para cada tutor. ·         El reporte debe poder visualizarse en pantalla antes de exportarse. ·         Los formatos de exportación permitidos son PDF y Excel. ·         La información mostrada en el reporte debe corresponder a los datos registrados mediante los casos de uso RF-52 y RF-55. |
