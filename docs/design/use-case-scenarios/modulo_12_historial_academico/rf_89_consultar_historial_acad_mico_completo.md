# RF-89: Consultar historial académico completo

| Campo | Descripción |
| :---- | :---- |
| **Autor** | Jessica Pablo Hernández |
| **Nombre** | RF-89: Consultar historial académico completo |
| **Actor** | Administrador, gestor administrativo |
| **Objetivo** | Permitir la consulta integral de la trayectoria académica de un alumno, mostrando toda la información histórica registrada en su expediente escolar para facilitar el seguimiento académico y administrativo. |
| **Flujo Principal** | **1\. Acceder al módulo de historial académico**  El actor autenticado selecciona el módulo **Historial Académico** desde el menú principal. El sistema valida que el usuario tenga permisos para consultar información histórica académica. **2\. Seleccionar opción de historial académico completo**  El actor selecciona la opción **Consultar historial académico completo**. El sistema muestra la pantalla de búsqueda de alumnos. **3\. Buscar y seleccionar expediente del alumno**  El actor busca al alumno por nombre o matrícula. El sistema consulta la base de datos, muestra los expedientes coincidentes y permite seleccionar el expediente correspondiente. **4\. Validar expediente académico**  El sistema verifica que el alumno exista, que el expediente académico esté disponible y que tenga información histórica asociada. **5\. Recuperar trayectoria académica completa**  El sistema consulta todos los registros académicos del alumno, incluyendo: ciclos escolares actuales y anteriores, calificaciones curriculares, evaluaciones cualitativas, talleres, clubes extracurriculares y cambios académicos registrados. **6\. Clasificar y organizar información académica**  El sistema separa y organiza los registros según su tipo académico (materias curriculares, talleres, evaluaciones descriptivas, clubes extracurriculares, cambios o movimientos académicos) y los agrupa por ciclo escolar, nivel educativo, periodo académico, materia y evaluación correspondiente. **7\. Mostrar historial académico completo**  El sistema presenta en pantalla toda la trayectoria académica del alumno, incluyendo calificaciones, evaluaciones, talleres, clubes extracurriculares y movimientos académicos históricos. **8\. Permitir navegación y consulta detallada**  El actor puede expandir o contraer ciclos escolares, consultar periodos específicos, revisar detalles de materias o regresar al listado de búsqueda **9\. Finalizar consulta**  El caso de uso termina cuando el actor cierra la vista del historial académico, regresa al menú principal o consulta otro expediente. |
| **Flujo Alterno** | **A1. Alumno no encontrado Ocurre en el paso<br>&nbsp;&nbsp;&nbsp;&nbsp;3. **<br>&nbsp;&nbsp;&nbsp;&nbsp;1. El sistema no encuentra expedientes coincidentes con el criterio de búsqueda.<br>&nbsp;&nbsp;&nbsp;&nbsp;2. El sistema muestra el mensaje: “No se encontraron alumnos con los datos ingresados”.<br>&nbsp;&nbsp;&nbsp;&nbsp;3. El actor puede modificar el criterio de búsqueda o cancelar la consulta.<br>&nbsp;&nbsp;&nbsp;&nbsp;4. Si modifica el criterio, el flujo regresa al paso<br>&nbsp;&nbsp;&nbsp;&nbsp;3. <br>&nbsp;&nbsp;&nbsp;&nbsp;5. Si cancela, el caso de uso termina sin mostrar información.  **A2. El alumno no tiene historial académico registrado Ocurre en el paso<br>&nbsp;&nbsp;&nbsp;&nbsp;5. **<br>&nbsp;&nbsp;&nbsp;&nbsp;1. El sistema no encuentra registros históricos asociados al alumno.<br>&nbsp;&nbsp;&nbsp;&nbsp;2. El sistema muestra el mensaje: “El alumno no cuenta con historial académico registrado”.<br>&nbsp;&nbsp;&nbsp;&nbsp;3. El actor puede regresar a la búsqueda o finalizar la consulta.<br>&nbsp;&nbsp;&nbsp;&nbsp;4. El caso de uso termina sin mostrar historial.  **A3. Existen cambios académicos registrados Ocurre en el paso<br>&nbsp;&nbsp;&nbsp;&nbsp;6. **<br>&nbsp;&nbsp;&nbsp;&nbsp;1. El sistema detecta movimientos académicos históricos, como modificaciones de calificaciones o cambios de inscripción.<br>&nbsp;&nbsp;&nbsp;&nbsp;2. El sistema incorpora la información en la trayectoria académica.<br>&nbsp;&nbsp;&nbsp;&nbsp;3. El sistema organiza los movimientos según el ciclo escolar y fecha correspondiente. |
| **Precondiciones** | El actor debe haber iniciado sesión en el sistema. El actor debe contar con permisos para consultar historial académico completo. El alumno debe estar registrado en el sistema. El expediente académico del alumno debe estar disponible. Deben existir registros académicos históricos o vigentes asociados al alumno. La base de datos académica debe estar disponible y accesible. |
| **Postcondiciones** | El historial académico completo del alumno queda visualizado en pantalla. La información histórica se muestra organizada y clasificada correctamente. No se modifica ningún registro académico. La consulta puede quedar registrada en bitácora según políticas institucionales. |
| **Reglas de negocio involucradas** | El historial académico debe incluir toda la trayectoria escolar registrada del alumno. La información debe organizarse por ciclo escolar, periodo y nivel educativo. Deben incluirse: calificaciones curriculares, talleres, evaluaciones cualitativas, clubes extracurriculares, movimientos académicos históricos. Solo usuarios autorizados pueden consultar historial académico completo. La consulta no debe modificar información académica. Las evaluaciones cualitativas deben mostrarse respetando su formato original. La información histórica debe pertenecer únicamente al alumno seleccionado. Las consultas pueden registrarse en bitácora institucional. |
