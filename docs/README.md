# Documentación del Proyecto

- `design/`: Documentos de diseño, diagramas de clases, casos de uso, estructura de base de datos y requerimientos. Estos documentos son mantenidos manualmente.
- `generated/`: Documentación generada automáticamente, como reportes de pruebas, migraciones de base de datos, documentación de la API y componentes de interfaz gráfica.
