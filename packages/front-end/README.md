# @sga/front-end

Capa de UI del Sistema de Gestión Académico.
- **Tecnologías**: Vite, React, TypeScript, TanStack Query, React Router, tRPC.
- **Estilos**: CSS Modules / Vanilla CSS (sin frameworks).
- **Consumo**: Consume exclusivamente los tipos de `@sga/data-access` y se comunica con el backend vía tRPC.
- **Renderizado**: WebView2 a través de Tauri.
